# NexusHR

**NexusHR** is an AI-Enabled Enterprise HR & Workforce Intelligence Platform built to streamline enterprise workforce management.

## Technology Stack

### Frontend
- React 19
- TypeScript
- Vite
- Tailwind CSS v4
- shadcn/ui
- TanStack Query
- React Router

### Backend
- Java 21
- Spring Boot 3.3
- Spring Security + JWT
- Spring Data JPA
- PostgreSQL
- Redis
- Flyway
- Spring AI + OpenAI

### DevOps
- Docker
- Kubernetes
- GitHub Actions

## Project Structure
- `/frontend`: The React 19 application.
- `/backend`: The Spring Boot 3.3 application.
- `/docs`: Project documentation and architecture details.
- `/docker`: Docker compose files and local environment setup.
- `/kubernetes`: Kubernetes manifests for deployment.
- `/scripts`: Utility scripts for database initialization and environment setup.
- `/.github/workflows`: CI/CD pipelines.
